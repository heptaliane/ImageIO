# ImageIO #
ImageIO is a library that provides simple image file I/O.

## Testing environment ##
### Arch Linux ###
* cmake 3.6.1
* gcc 6.1.1

## Supported Image format ##
### Bitmap Images ###
* OS/2 bitmap (1, 4, 8, 24, 32 bit)
* uncompressed Windows bitmap (1, 4, 8, 24, 32 bit)

## Build ##
```bash
cd build
cmake ../
make
```
